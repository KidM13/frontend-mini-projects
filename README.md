
# Frontend Mini Projects

A collection of small frontend projects built using **HTML, CSS, and JavaScript**. This repository is created **for learning, practice, and skill-building purposes**, focusing on real-world UI components, logic implementation, and core frontend concepts.

## Projects Included

* Warm-up UI exercises
* Interactive Quiz App
* Smart To-Do Application
* More mini-projects added progressively

## Goals

* Strengthen core frontend fundamentals
* Practice DOM manipulation and event handling
* Improve UI structuring and layout skills
* Build real-world interactive components

## Requirements

* Modern web browser
* Basic understanding of HTML, CSS, and JavaScript

## How to Run

1. Clone or download the repository.
2. Open any project folder.
3. Open the `index.html` file in your browser.

## Purpose

This repository is built purely for **learning, experimentation, and continuous improvement in frontend development.**
