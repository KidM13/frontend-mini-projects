# Expense Tracker

## Description
A simple expense tracking web application built for learning and understanding basic frontend development concepts. 
The project allows users to add expenses, view them in a list, calculate the total amount spent, and remove entries. It is designed as a small practice project to improve DOM manipulation and event handling skills.

## Features
- Add a new expense with description and amount
- Display a list of recorded expenses
- Automatically calculate and update the total spent
- Delete individual expenses

## Technologies
- HTML
- CSS
- JavaScript

## Requirements
- A modern web browser
- A code editor such as Visual Studio Code (optional)

## How to Run
1. Clone the repository
2. Navigate to the `expense-tracker` project folder
3. Open the `index.html` file in a web browser

## Purpose
This project is created for educational purposes to practice JavaScript DOM manipulation, event handling, and basic frontend project structure.
