
# Interactive Quiz App

A simple frontend quiz application built using **HTML, CSS, and JavaScript**.
It dynamically displays questions, handles user answers, and shows feedback instantly.

## Features

* Multiple choice questions
* Dynamic question loading
* Instant feedback (correct / wrong)
* Automatic next question loading
* Clean and simple UI

## Purpose

This project is built to practice:

* DOM manipulation
* Event handling
* Dynamic UI rendering
* Basic state management in JavaScript

## Technologies Used

* HTML
* CSS
* JavaScript

## How It Works

Questions are stored in a JavaScript array and rendered dynamically.
Each click updates the UI based on user interaction and quiz state.
